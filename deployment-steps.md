# Deployment Steps

1. Created a custom VPC for the project.
2. Added public and private subnets inside the VPC.
3. Configured route tables and attached an Internet Gateway.
4. Launched EC2 instances inside the private subnets.
5. Set up an Application Load Balancer to handle incoming traffic.
6. Created an Auto Scaling Group with CPU-based scaling rules.
7. Stored static assets and backup files in Amazon S3.
8. Tested scaling behavior using AWS lab scenarios.
